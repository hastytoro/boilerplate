The font we using is Inter.

Right click the optimized (hashed) file within the DevTools and choose to Open in new tab to download the actual font file. Now copy and rename it into your local project /public/font folder.

Optionally we can implement a self-hosted font. Google Fonts has done all the hard work for us, optimizing the font. We can rather serve it locally, saving on request(s) that would occur.

Now we use the @font-face at-rule that specifies a custom font. The font can be loaded from either a remote server or in this case locally-installed.
