Flow layout is required/supported ✅

No matter how long this paragraph is, only 3 lines will be shown, the rest concealed, and an ellipsis character will be added to demonstrate that there is more text that is not shown here.

Short paragraphs don't get an ellipsis.

Flex items (children not-supported) ❌

Warning: In certain cases the -webkit line-clamp can appear buggy. This can happen when the element we're applying these styles to is also used as a child flex item in the Flexbox Layout capacity.

Here we avoid the issue as we line clamp our flex item but it isn't being stretched or flexed as part of a Flexbox or Grid Layout because we use an enclosing wrapper.

Flexbox child is a single Flow layout container ✅

Direct children that are flex item(s) inside a Flexbox parent will be stretched according to the layout algorithm. We need our clamped paragraphs to use Flow layout so we wrap our paragraph in a div.
