Like flex-grow the fr unit is flexible. Our first column wants to take up 1/3rd of available space, but its child could be wider and so would grow to accommodate it.

The grid reserves 200px for the first column. Then it distributes whatever space remains to the other 2. This is how flex-grow works in Flexbox when we don't set flex-basis: 0.

* fr distributes available space.

NOTE: This behaviour only happens with fr unit and auto but px, rems and % values are all hard limits.
